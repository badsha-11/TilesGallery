# Tiles Gallery – Premium Interior Solutions

**Tiles Gallery** is a modern, luxury-focused web application designed to provide users with a refined browsing experience for premium marble and ceramic tiles. The platform functions as a dynamic catalog, allowing users to explore and evaluate high-quality tile collections with ease.

## Live Application

https://tiles-gallery-chi.vercel.app/

---

## Project Objective

The primary goal of this project is to deliver a premium user interface that enables customers to efficiently discover suitable tile options for their interiors. The application emphasizes performance, usability, and a polished user experience built on a modern development stack.

---

## Key Features

* **Dynamic Tile Gallery**
  Tiles are dynamically loaded from a structured `data.json` file and displayed in a clean grid layout.

* **Real-time Search**
  Users can quickly find tiles by searching through names or categories.

* **Smooth Animations**
  Integrated animations enhance the interface, providing seamless transitions and improved visual engagement.

* **Detailed Product Pages**
  Each tile includes a dedicated detail page with specifications and a refined presentation.

* **Review Slider**
  A continuous, looping slider showcases customer feedback and enhances credibility.

* **Responsive Design**
  Fully optimized for mobile, tablet, and desktop devices.

* **Modern UI Design**
  A glass-inspired interface with a subtle glowing theme ensures a contemporary look and feel.

---

## Technologies Used

This project is built using the following technologies:

* **Next.js** – Core framework for building the application
* **Tailwind CSS** – Utility-first styling and responsive design
* ** MongoDB** – Database for storing tile data and application content
* **Framer Motion** – Animation and transition handling
* **Swiper.js** – Carousel and slider functionality
* **React Icons** – Icon library for UI elements
* **Lucide React** – Clean and modern icon set

## Local Setup

1. Clone the repository:

```bash
git clone https://github.com/ishrankhan6111-8883/tiles-gallery.git
```

2. Navigate to the project directory:

```bash
cd tiles-gallery
```

3. Install dependencies:

```bash
npm install
```

4. Run the development server:

```bash
npm run dev
```

5. Open your browser and visit:

```
http://localhost:3000
```
